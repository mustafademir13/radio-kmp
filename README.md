# RadioKMP

Kotlin Multiplatform + Compose tabanlı radyo uygulaması başlangıç projesi.

## İlk hedef
- Android'de canlı radyo stream oynatma
- Sonraki adım: iOS actual player + kanal listesi + arka plan medya kontrolleri

## Çalıştırma
1. Android Studio ile bu klasörü aç
2. Gradle sync tamamlanınca `composeApp` konfigürasyonunu çalıştır
